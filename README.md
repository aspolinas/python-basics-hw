# Python Basics — Data Analysis

Простой скрипт для базового анализа числовых данных на Python.

## Что делает

- `analyze_numbers(numbers)` — считает количество, сумму, среднее, минимум и максимум для списка чисел
- `filter_above_mean(numbers)` — возвращает числа, которые выше среднего

## Запуск

```bash
python main.py
```

## Пример вывода

```
Input data: [23, 45, 12, 67, 34, 89, 2, 56, 78, 41]
  count: 10
  sum: 447
  mean: 44.7
  min: 2
  max: 89

Above mean (44.7): [45, 67, 89, 56, 78]
```
